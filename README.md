## Geral
Nome: Radio Browser
Descrição: App web para escutar rádios e salvar as favoritas para escutar quando quiser.
Tecnologias: NextJS, TailWind e Flowbite-react.

## Como instalar
Primeiro, rode o comando do servidor:

```bash
npm install
npm run dev
# or
yarn install
yarn dev
```

Desafio proposto pela Coodesh.
